# CloudOps

CloudOps is a library for Android that simplifies file operations for file hosting services.


## Requirements

+ Android 4.3 and up


## Supported services

+ ownCloud
+ Nextcloud


## Used by

+ [ownRun](https://github.com/Alkisum/ownRun)
+ [Notepad](https://github.com/Alkisum/Notepad)
